nouveau-front-populaire-legislatives-2024.fr est une initiative citoyenne, non-officielle et indépendante de soutien aux campagnes du Nouveau Front Populaire, visant à s'assurer que chaque candidat·e a bien une page web pour visibiliser sa campagne. Si une page web dédiée à la campagne existe déjà, merci de nous en informer pour que la présente page redirige vers celle-ci. Si vous avez des notions de HTML, vous pouvez contribuer à améliorer cette page directement depuis son code source. Si ce n'est pas votre cas, le plus simple est encore de nous écrire.
